# Anathi Letsela | Portfolio Website

This is my **personal portfolio website**, showcasing my skills, projects, and contact information.  
Built using **HTML and CSS**, it is designed to provide a professional online presence for potential employers and learnership applications.

## 🔹 Features

- Introduces **Anathi Letsela** and career objectives  
- Lists technical **skills** including HTML, CSS, JavaScript, and Full Stack Development  
- Showcases **projects** with live demo links

- Clean, responsive, and professional design
